# Changelog

## [Current Version]

### Features Added
- **Provenance Tracking**: Transparent source attribution with supporting snippets and confidence scores
- **Advanced Retrieval Processing**: Semantic re-ranking, deduplication, and freshness filtering
- **Web Dashboard**: Interactive UI with real-time progress tracking and organized result tabs
- **Multiple AI Providers**: Support for NVIDIA, OpenAI, Fireworks AI, OpenRouter, and local models
- **API Server**: REST API for integration with other applications

### Technical Improvements
- Enhanced search result quality through ML-based processing
- Real-time progress tracking with Server-Sent Events
- Comprehensive documentation and examples
- Docker support with compose configuration
- Graceful error handling and fallback mechanisms

### Benefits
- **Transparency**: See exactly where information came from
- **Quality**: Higher relevance and less redundant results  
- **Usability**: User-friendly web interface
- **Integration**: Easy API access for developers
- **Reliability**: Multiple fallback options and error handling
